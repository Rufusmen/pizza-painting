gameloop
read rows:int columns:int
loop rows read board:word(columns)
read entities:int
loop entities read id:int paint:int x:int y:int
loop entities loop 3 read pawnColor:word(3)
write MOVE 0 0

STATEMENT
Paint more cells than your opponent

INPUT
rows: Rows of board
columns: Columns of board
board: Board description
entities: Number of your pawns
id: Id of pawn
x: x coordinate of pawn centre
y: y coordinate of pawn centre
paint: units of paint on pawn
pawnColor: description of pawn coloring

OUTPUT
MOVE id direction | SHOOT id direction range
